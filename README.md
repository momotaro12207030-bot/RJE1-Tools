# 作業時間・HC計算ツール

FCST（予測物量）とBacklog数量から、必要な作業時間（LH）と人員（HC）を自動計算するシンプルなWebツールです。ブラウザのみで動作し、インストール不要で軽快に使用できます。

このツールは、現場での人員判断や作業負荷の把握を素早く行うことを目的としています。入力した数値はすべてブラウザ内で処理され、サーバー通信やデータ保存は行いません。

対象工程は以下の4つです。

1. Check In  
2. IDRT  
3. Case Stow  
4. Case Receive  

各工程ごとに以下の項目を入力します。

・FCST（ケース数）  
・Backlog（ケース数）  
・平均処理能力（ケース/時）  

入力値から、必要作業時間（LH）と必要人員（HC）を自動算出します。

計算ロジックは次の通りです。

必要時間（LH）
LH = (FCST + Backlog) ÷ 処理能力（case/h）

処理能力が0以下の場合は、計算不可のためLHは0として扱います。

必要HC
HC = LH ÷ 8

1人あたり8時間シフトを前提としています。

画面右側には、パレット数からBacklogを算出する入力パネルを用意しています。

Backlog(case) = パレット数 × ケース/パレット

ここで計算された値を「反映」ボタンで各工程のBacklog欄へ自動入力できます。

画面下部にはサマリーを固定表示します。

・合計LH  
・合計HC  
・不足LH（出勤LH − 必要LH）  
・不足HC（出勤HC − 必要HC）  

不足が発生している場合は、数値が赤文字で表示されます。

使用手順は以下の通りです。

1. 各工程のFCSTを入力する  
2. 平均処理能力を入力する  
3. 右側のBacklogパネルでパレット数とケース数を入力する  
4. 「反映」ボタンを押してBacklogへ反映する  
5. 出勤HCを入力する  

入力内容に応じて、必要LH、必要HC、不足量が自動更新されます。

このツールの特徴は次の通りです。

・完全ブラウザ内計算でサーバー不要  
・データ保存なしで安全  
・静的HTMLのため高速動作  
・スマートフォン対応レスポンシブUI  
・誤操作で数式が壊れることがない設計

セキュリティについて

本ツールはサーバー通信、ログイン、データ保存を行わない静的ページのため、外部からの書き換えやデータ漏洩のリスクはほぼありません。入力された数値はすべて利用者のブラウザ内でのみ処理されます。
